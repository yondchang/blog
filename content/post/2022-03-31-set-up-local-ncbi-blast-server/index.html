Step 1: Log on to your ECS in the cloud

Step 2:

install Docker following instruction here https://docs.docker.com/engine/install/ubuntu/

docker fetch wurmlab/sequenceserver

makeblastdb -dbtype nucl -title Organism -in path_for_library/Organism.fasta -parse_seqids

docker run --rm -itp 4567:4567 -v path_for_library:/db wurmlab/sequenceserver
